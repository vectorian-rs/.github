# vectorian-rs

Open-source Rust tools for performance, observability, and developer productivity.

## Projects

| Project | Description |
|---------|-------------|
| [harrow](https://github.com/vectorian-rs/harrow) | Macro-free HTTP framework over Hyper with tokio, monoio, and io_uring backends |
| [rolly](https://github.com/vectorian-rs/rolly) | Lightweight observability — hand-rolled OTLP protobuf over HTTP, built on tracing |
| [spinr](https://github.com/vectorian-rs/spinr) | HTTP performance and debugging tool with trace, load test, and MCP server modes |
| [wrk3](https://github.com/vectorian-rs/wrk3) | HTTP benchmarking with constant-throughput and correct-latency measurement (wrk2 fork) |
| [chizu](https://github.com/vectorian-rs/chizu) | Subject-to-file routing for coding agents — local repository knowledge graph |
| [qrst](https://github.com/vectorian-rs/qrst) | On-device hybrid search (BM25 + vector + fusion) for markdown, code, and web components |
| [rie](https://github.com/vectorian-rs/rie) | Distributed ONNX embedding service with router and worker architecture |
| [marie-ssg](https://github.com/vectorian-rs/marie-ssg) | Fast static site generator — markdown with TOML metadata to HTML |
| [kiters](https://github.com/vectorian-rs/kiters) | Rust utilities: timestamps, request IDs, and external IDs |
| [local-rs](https://github.com/vectorian-rs/local-rs) | Reverse proxy combining static file serving with API proxying |
| [ferrview](https://github.com/vectorian-rs/ferrview) | Lightweight system monitoring with node agents and a collector dashboard |
| [edgeview](https://github.com/vectorian-rs/edgeview) | CloudFront log analytics with SVG/HTML reports |
| [nacre](https://github.com/vectorian-rs/nacre) | Local-first web dashboard for the beads issue tracker |
